What's the difference?

HTML

Hypertext Markup Language

Create the stucture

  • Creates the layout of the content
  • Provides the structure for the webpage design
  • The fundamental building block of any webpage

CSS

Cascading Style Sheet

Stylize the website

  • Applies style to the web page elements
  • Targets various screen sizes to make the webpages responsive
  • Primarily handles the "look and feel" of a web page

Javascript

Increase interactivity

  • Adds interactivity to a webpage
  • Handles complex functions
  • Programmatic code which enhances functionality
